# smart-wearable-glove
An ESP32-based smart wearable glove designed for real-time gesture recognition, speech conversion, and assistive communication. The system uses sensors to detect hand gestures and converts recognized gestures into meaningful commands and speech, helping improve communication for people with speech and hearing impairments.
